Windows 11에서 변경된 고대비 모드 소개

안녕하세요, 엔비전스입니다.

더운 여름이 지나가고, 가을이 찾아오고 있습니다. 올 여름, 지난 6월에는 Windows 11 출시가 발표되었습니다.

Windows 11은 6년만에 출시되는 Microsoft의 차기 운영체제입니다. 정말 오랫동안 Windows 10을 꾸준히 업데이트해 왔는데요.

이번 Windows 11 접근성 기능 중에 가장 크게 바뀐 것은 고대비 모드라고 할 수 있겠습니다. 이번 시간에는 간단하게 고대비 테마를 한 번 살펴보도록 합시다.

Windows 11의 고대비 모드는 크게 네 가지 테마를 가지고 있습니다. 지금부터 하나씩 구경하러 가 보시지요.

사진 표시기

라디오 버튼으로 원하는 사진을 표시하세요.

수생(Aquatic)

푸른 빛이 도는 매우 어두운 회색을 기본 바탕으로 하는 수생 고대비 테마입니다. 이번 Windows 고대비 테마는 대비만 높은 것이 아니라, 오랫동안 보기 편한 색상, 조금 더 단조롭지 않은 새로운 디자인을 추구한 것 같습니다.

사막(Desert)

기존의 White 테마를 대체하는 테마입니다. White 고대비 테마는 밝은 바탕에 어두운 글씨를 좋아하는 사람일지라도, 대비가 너무 극명하게 높다보니, 오래 보면 눈이 아프다는 단점이 있었습니다. 이번에 추가된 고대비 테마들은 대비를 약간 낮춘 대신, 눈의 피로를 줄이려고 한 것 같습니다.

약간의 노란빛이 도는 흰색, 베이지색이나 상아색 계통의 바탕을 사용해서, 상대적으로 눈이 덜 부신걸 알 수 있습니다.

황혼(Dusk)

위에서 설명한 수생과 비슷하면서 다-른 테마입니다. 수생보다 더 무채색에 가까운 배경색을 사용하며, 한층 더 밝은 느낌을 줍니다. 너무 어두운 화면이 싫은 사람에게 이 ‘황혼’ 테마가 많은 사랑을 받을 것 같습니다. 언뜻 보기에는 다크모드를 적용한 것 처럼 보이기도 하네요.

밤하늘(Night Sky)

말은 밤하늘이라고 적어놓았지만, 아마 클래식이라는 명칭을 붙혀도 어울리지 않을까 싶은 테마입니다.

가장 흔히, ’고대비’라는 이미지 하면 떠오르는 테마로, 검정 바탕에 흰색 또는 노랑색 등의 글씨를 사용한 테마입니다.

밤하늘 테마는 기존 Black 테마라고 보아도 무방할 것 같습니다. White 테마가 사라진 것에 반해, Black 테마는 전통적으로 저시력자 사이에서 사랑받던 고대비 테마로 ‘밤하늘’ 이라는 명칭을 얻었습니다.

설정 방법(Windows 11 Insder Preview)

Windows 11(한글 기준) 현재 프리뷰 버전에서는 고대비 테마, 고대비 모드의 명칭이 “대비 테마”로 변경되었습니다.

Windows 11 Insider Preview가 깔려있다면 아래와 같이 설정할 수 있습니다.

새로운 고대비 테마의 테마 설정은 설정 앱에서 가능하며, 단순한 일반 테마와의 토글은 이전과 동일한 Alt + Shift + Print Screen을 통해 할 수 있습니다.

고대비 테마를 위해 우리가 대비해야 하는 것

다양한 OS 환경설정에 따라 웹 환경도 맞춰서 변화하고 있습니다. 최근에는 많이 대중화된 prefers-color-scheme을 통한 다크모드 지원만 보더라도 알 수 있습니다.

이런 고대비 모드는 prefers-color-scheme처럼 선택적으로 지원해야 하는 것이 아닌 반드시 대응해야만 사용자가 불편함없이 웹사이트를 탐색할 수 있게 됩니다.

이번 대비 테마는 Windows의 기능이기는 하지만, 이전처럼 Microsoft 브라우저에서만 지원하는 기능이 아닙니다. Edge뿐만아니라, Chrome이나 Firefox에서도 적용하고 있습니다. Windows에 종속된 기능이지만, 많은 브라우저에서 이에 대한 대응을 지원하는 만큼 널리 사용될 것입니다.

위 테마 사진들을 잠깐 예로 들어보자면, 어두운 테마에서 NULi 사이트를 방문했을 때, NULi 로고가 안 보이는 부작용이 있습니다. PNG 파일이기 때문에, 투명한 배경에 갈색 글씨로 로고가 쓰여져 있는 로고여서 밝은 화면으로 봤을 때는 아주 눈에 잘 뛰지만 이렇게 어두운 고대비 테마를 적용하면 잘 안보이게 됩니다.

로고나, 아이콘 등에는 SVG의 사용이 더더욱 필요해지게 되었고, 우리는 이에 맞춰 대응해야 합니다. 물론 SVG가 아니라, 로고의 색상을 테마에 맞게 여러개 준비해두거나 필터를 사용하는 것도 방법이 되겠네요. 이에 대한 대응방법에 대해서는 다음 시간에 더욱 심도있게 다를 수 있도록 준비할 예정입니다.

이번 시간에는 짧게 Windows 11의 새로운 대비 테마를 살펴봤습니다. 유익한 정보가 되셨으면 합니다. 앞으로도 새로운 소식을 갖고 찾아뵙겠습니다. 감사합니다.
